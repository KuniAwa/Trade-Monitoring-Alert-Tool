# フォルダ階層 → Excel 出力（簿記分類付き）

指定フォルダ内のファイル名・フォルダ名と階層を自動で抽出し、**ファイル名から簿記の6分類を推定**してExcel（またはCSV）に出力します。分類はExcel上で**事後的に変更可能**（ドロップダウンで選択）です。

**ツールの格納場所:** `C:\Users\kunik\OneDrive\ドキュメント\Apps\Cursor\フォルダの情報の抽出`

## 準備

```powershell
cd "C:\Users\kunik\OneDrive\ドキュメント\Apps\Cursor\フォルダの情報の抽出"
pip install openpyxl
```

または

```powershell
cd "C:\Users\kunik\OneDrive\ドキュメント\Apps\Cursor\フォルダの情報の抽出"
pip install -r requirements.txt
```

## 使い方

- **カレントフォルダ**を対象にする場合:
  ```powershell
  cd "C:\Users\kunik\OneDrive\ドキュメント\Apps\Cursor\フォルダの情報の抽出"
  python folder_to_excel.py
  ```

- **特定のフォルダ**を指定する場合:
  ```powershell
  cd "C:\Users\kunik\OneDrive\ドキュメント\Apps\Cursor\フォルダの情報の抽出"
  python folder_to_excel.py "C:\対象のフォルダパス"
  ```

- **出力ファイル名**を指定する場合:
  ```powershell
  python folder_to_excel.py "C:\対象フォルダ" -o "出力.xlsx"
  ```

- **Excelが使えない環境**では CSV で出力:
  ```powershell
  python folder_to_excel.py --csv
  ```

## 簿記の6分類

| 分類     | 説明     |
|----------|----------|
| ①資産   | 現金・売掛・在庫・固定資産など |
| ②負債   | 借入・買掛・未払・前受など     |
| ③資本   | 資本金・繰越利益・剰余金など   |
| ④収益   | 売上・収益・雑収入など         |
| ⑤費用   | 仕入・人件費・経費・調査費など |
| ⑥その他 | 上記に当てはまらないもの       |

ファイル名や親フォルダ名に含まれるキーワードから自動で推定します。**Excelの「分類」列はドロップダウンから選び直せるため、事後的に修正できます。**

## 出力内容

| 列名       | 内容                                         |
|------------|----------------------------------------------|
| 階層レベル | ルートからの深さ（1, 2, 3…）                 |
| 親フォルダ | その項目の親フォルダの相対パス               |
| 名前       | ファイルまたはフォルダ名                     |
| 種類       | 「ファイル」または「フォルダ」               |
| **分類**   | **①資産～⑥その他（推定値・Excelで変更可）** |
| 拡張子     | ファイルの拡張子（例: .txt）                 |
| サイズ(バイト) | ファイルのバイト数（フォルダは空欄）     |
| サイズ(表示)  | 読みやすい単位（例: 1.5 MB、200 KB）     |
| 作成日時   | ファイル・フォルダの作成日時                 |
| 更新日時   | 最後に更新された日時                         |
| フルパス   | 絶対パス                                     |

出力ファイルは、指定したフォルダ（またはカレントフォルダ）に `folder_structure_日時.xlsx` の名前で保存されます。
