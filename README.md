# Apps / Cursor ワークスペース

このリポジトリには複数のプロジェクトが含まれます。

| フォルダ | 内容 |
|----------|------|
| [**market-alert/**](market-alert/README.md) | 相場監視アラート（Vercel + Twelve Data / Yahoo）。`api/cron`、Cron メール通知。 |
| **web/** | Next.js など別アプリ |
| その他 | `english-learning/`、`ifrs_gaap_comparison/`、`フォルダの情報の抽出/` など |

**相場監視ツールをデプロイするとき**は、Vercel の **Root Directory** を **`market-alert`** に設定してください。
